# Advanced Calculator

A modern and interactive calculator built using HTML, CSS, and JavaScript with DOM Manipulation.

## Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)

- Advanced operations
  - Percentage (%)
  - Square (x²)
  - Cube (x³)
  - Power (xʸ)
  - Square Root (√)
  - Factorial (n!)
  - Logarithm (log)
  - Pi (π)

- Trigonometric functions
  - Sin
  - Cos
  - Tan

- Additional Features
  - Calculation History
  - Clear History
  - Delete Last Character
  - Dark/Light Theme Toggle
  - Keyboard Support
  - Responsive Design
  - Attractive UI with Animations

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
